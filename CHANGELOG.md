## 0.1.0 - 2026-06-17

### Added

- Created the initial static browser prototype.
- Added 24-base-pair slime genomes with seeded procedural gene mapping.
- Added clickable ASCII DNA editing with realistic base pairing.
- Added slime synthesis, visible trait descriptions, and estimate ranges.
- Added timed tests for discovering and refining trait information.
- Added breeding and crossbreeding between mature samples.
- Added lifespan, maturity, living sample handling, and release/contain controls.
- Added scientist vitals, skills, XP gain, stamina costs, regeneration, and rest tasks.
- Added a command-style XP cheat input for testing skill progression.
- Added a collapsible time queue drawer with skip controls.
- Added automatic, manual, and disabled journal modes.
- Added local save, import/export, and export-to-folder support.
- Added visual polish for base coloring, seed placement, and trait markers.
